# PairChecker - Проверка парных файлов

Приложение для проверки наличия парных файлов разных форматов в указанной папке.

## Внесенные улучшения

### 1. **Архитектура - Разделение ответственности**
- ✅ Создан отдельный класс `FileChecker` для бизнес-логики
- ✅ MainForm теперь отвечает только за UI
- ✅ Код стал более тестируемым и поддерживаемым

### 2. **Обработка ошибок**
- ✅ Улучшена обработка исключений при работе с файловой системой
- ✅ Ошибки собираются и отображаются в отчете
- ✅ Добавлены информативные сообщения об ошибках

### 3. **Производительность**
- ✅ Использование `EnumerateFiles` вместо `GetFiles` (меньше памяти)
- ✅ Корректная обработка отмены операции через CancellationToken
- ✅ Обработка ошибок доступа к отдельным файлам без прерывания всей операции

### 4. **Удобство использования**
- ✅ Добавлено сохранение настроек приложения (последняя папка, форматы, режим)
- ✅ Настройки автоматически загружаются при запуске
- ✅ Улучшены сообщения о прогрессе

### 5. **Качество кода**
- ✅ Устранено дублирование кода
- ✅ Улучшена читаемость
- ✅ Добавлены XML-комментарии
- ✅ Следование принципам SOLID

## Основные классы

### FileChecker
Отвечает за:
- Поиск файлов по форматам
- Анализ парных файлов
- Генерацию отчетов
- Уведомление о прогрессе

### AppSettings
Отвечает за:
- Сохранение настроек пользователя
- Загрузку настроек при запуске
- Хранение последних использованных значений

### MainForm
Отвечает за:
- Отображение UI
- Взаимодействие с пользователем
- Координацию работы компонентов

## Использование

1. Выберите папку для проверки
2. Укажите два формата файлов для сравнения
3. Включите рекурсивный поиск при необходимости
4. Нажмите "Запустить проверку"
5. Сохраните отчет при необходимости
